# Code of Conduct

## Our Pledge

We want AskUser Popup to be a practical and respectful project for people building human-in-the-loop agent tooling. Contributors are expected to communicate with care, assume good faith, and keep criticism focused on the work.

## Expected Behavior

- Be respectful and direct.
- Prefer technical evidence over personal remarks.
- Accept maintainers' decisions about scope and project direction.
- Report harmful behavior privately when possible.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks.
- Publishing private information without consent.
- Repeated disruption after maintainers ask for a topic to stop.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block accounts that violate this code of conduct.
